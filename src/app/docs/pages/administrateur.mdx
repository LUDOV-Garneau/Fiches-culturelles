# Doc administrateur
---
## Page Admin

La Page Admin permet aux gestionnaires de l’application de consulter, organiser et administrer l’ensemble des jeux présents dans la base de données.
Elle offre une interface centralisée pour **visualiser, sélectionner, modifier et supprimer** des jeux. 
Cette page permet à l'administrateur de **contrôler** l'affichage de chaque élément de la page principale.

## Connexion

Pour accéder à la page d'administration, l'utilisateur doit appuyer sur le bouton « Connexion » situé dans le coin supérieur droit de la page.
En suite, il doit rentre le courriel et le mot de passe associe à la compte administrateur et clicker sur le button "Se connecter" pour être redirectionné à la page avec la liste des jeux.

## Liste des jeux

Sur la page de la liste des jeux, l’administrateur peut consulter tous les jeux importés depuis la plateforme KOHA. Il peut également sélectionner les jeux affichés dans le carrousel, modifier certaines informations propres à chaque jeu et supprimer des jeux de la base de données locale (sans affecter les données sur KOHA).
L’administrateur peut revenir à la page principale en appuyant sur le bouton situé en haut de la page portant la mention "Retourner".

### Selectionné carroussel

**Ajout un jeu au carrousel**: Si un jeu n’est pas sélectionné dans le carrousel, appuyez sur le bouton vert  "Ajouter" sur la carte du jeu souhaité.
**Retirer un jeu du carrousel**: Si vous souhaitez retirer un jeu du carrousel, appuyez sur le bouton jaune "Retirer" sur la carte du jeu souhaité.
